Les Protéines

Les bâtisseurs de notre organisme.

Réparation et Énergie

Les protéines sont les nutriments qui permettent la réparation de nos muscles. Composées d'acides aminés, elles constituent une source d'énergie métabolisée plus lentement que les glucides, mais plus rapidement que les lipides. Elles sont essentielles pour assurer le bon fonctionnement du corps, car elles garantissent littéralement que nos fibres musculaires ne s'érodent pas.

Sources Animales : Les protéines "complètes"

Contrairement aux idées reçues, on trouve des protéines aussi bien dans le règne animal que végétal. La nuance est que les protéines animales sont dites complètes : elles contiennent tous les acides aminés permettant au corps de les assimiler entièrement pour ses besoins.

On les trouve dans : les produits laitiers, les yaourts, la viande ou encore le poisson.

Sources de protéines végétales : L'art des combinaisons

Les protéines végétales ont souvent des proportions d'acides aminés différentes du ratio idéal. Il faut donc combiner les sources (comme les céréales et les légumineuses) pour que le corps les assimile efficacement.

On en trouve dans les haricots, pois chiches, soja, lentilles ou cacahuètes. La levure de bière est aussi un excellent complément car elle contient souvent les acides aminés qui manquent aux autres sources végétales.

Le saviez-vous ?

Les oléagineux (amandes, noix de cajou, graines de tournesol ou de sésame) sont d'excellentes sources de protéines, mais aussi de micronutriments précieux pour la santé.
